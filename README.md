
# 🎮 Lumina Ecosystem

> ⚡ A powerful brazillian open-source Discord + League of Legends ecosystem built for gamers, developers, and communities.

---

## 🌟 What is Lumina?

**Lumina** is a full-stack gaming ecosystem combining:

- 🤖 Discord Bot (League of Legends assistant + moderation)
- 🚀 REST API (data, auth, integrations)
- 🌐 Web Dashboard (modern UI for control & stats)

Built for **communities that love League of Legends, automation, and clean systems.**

---

```
👾 Discord User
     │
     ▼
🤖 Lumina Bot
```

## 🤖 Lumina Bot

Your Discord companion for everything about League of Legends.

### ⚔️ Features

- 🎮 Summoner stats & profiles
- 🏆 Match history tracking
- 🎯 Champion rotation updates
- 🛡️ Moderation system (kick/ban/mute)
- 📊 Advanced analytics & charts
- 🔔 Smart notifications
- ⚙️ Permission system (roles + staff tiers)

### 🧰 Tech Stack

- Discord.js v14
- Node.js 20+
- Canvas / Chart.js
- MongoDB
- Riot Games API

---

## 🚀 Lumina API

The brain of the ecosystem.

### 🔥 Features

- 🔐 Discord OAuth2 authentication
- 🧾 JWT secure sessions
- 🎮 League of Legends data aggregation
- 👤 User inventory system (skins, stats)
- 🧠 Internal bot communication
- 📊 Logging & monitoring system

### 🧰 Tech Stack

- Node.js + Express
- MongoDB (Mongoose)
- JWT
- OAuth2 Discord
- Riot Games API

---

## 🌐 Lumina Dashboard

A modern control panel for the entire ecosystem.

### 🎨 Features

- 🎮 Player stats visualization
- 🎨 Skin inventory system
- 📊 Interactive charts
- 👤 Discord login integration
- ⚙️ Bot configuration panel
- 🔍 Smart filtering system
- 📱 Fully responsive UI

### 🧰 Tech Stack

- React 18
- Vite
- Tailwind CSS
- OAuth2 PKCE
- React Router

## 🔗 How everything connects

- 🤖 Bot handles Discord interaction

```bash
npm install
```
